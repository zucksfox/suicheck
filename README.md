
# Wallet checker balance




## Cara Install

Buka terminal kamu + Download dulu

```bash
  git clone https://github.com/zucksfox/suicheck.git
```
Buka Folder kamu
```
cd suicheck
```
Run
```
python suicheck.py 
```
